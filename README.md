# InteliSecure

AI-Powered Cybersecurity Threat Detection, Analysis, and Incident Response System

## Project Status

### Completed Features
- FastAPI project foundation initialized
- SQLite database with SQLAlchemy ORM integrated
- Database models for Users, Logs, and Alerts
- User Authentication (JWT, bcrypt, RBAC)
- Login Logger and Multi-source Log Parsers
- API Routes for Auth and Logs
- Frontend UI dashboard skeleton (HTML/CSS/JS)
- Plotly chart integration
- Basic directory structure established

### Missing Features
- Brute Force, Port Scan, and Unauthorized Access Detectors
- Real-Time Monitor System
- Threat Routes API
- Explainable AI (Threat Explainer)
- Security Score Engine and API
- Recommendation Engine
- ML Training and Prediction pipelines
- Device Discovery and LAN Monitor
- Dashboard Live API Integration
- PDF and PPT Report Generators
- Glossary and Learning Helper
- Unit Tests and Attack Simulation
